    Instrucciones para usar la página

    1. Definir Lenguaje: En el primer campo, ingresa la expresión que define el lenguaje del autómata, por ejemplo: "a^n b^n".

    2. Cadena de Entrada: Ingresa una cadena para procesar, por ejemplo: "aabb".

    3. Generar Autómata: Haz clic en el botón "Generar Autómata" para crear el autómata de pila según los parámetros ingresados.

    4. Procesar Cadena: Haz clic en "Procesar" para analizar la cadena y ver cómo el autómata funciona con ella.

    5. Visualización: Aquí podrás ver el estado actual, la pila del autómata y las transiciones aplicadas.
